# zed-scss

SCSS support for [Zed](https://zed.dev).

## Features

- Syntax Highlighting
- Outline View
- IntelliSense(uses vscode-css-language-server and [some-sass](https://github.com/wkillerud/some-sass/))
- Formatting

## Grammar

- [tree-sitter-scss](https://github.com/savetheclocktower/tree-sitter-scss)
